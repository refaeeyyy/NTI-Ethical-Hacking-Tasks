## Overview
This repository contains documented cybersecurity lab exercises completed as part of a supervised academic training program at NTI.

All activities were conducted in isolated lab environments for educational and defensive security understanding only.

## Topics Covered
- Information Gathering & OSINT Techniques
- Network Traffic Analysis using Wireshark
- Windows Security Concepts and Controlled Exploitation Scenarios
- Man-in-the-Middle Attack Simulations for Risk Awareness
- Malware Analysis for Detection and Prevention Understanding
- IoT Malware Research (Mirai Botnet Case Study)

## Learning Objectives
- Understand common attack techniques to improve defensive security strategies
- Analyze network traffic to identify suspicious behavior
- Study malware behavior to enhance detection and mitigation methods
- Map attack techniques to security controls and MITRE ATT&CK framework

## Skills Demonstrated
- Information Gathering & Reconnaissance
- Network Traffic Analysis (Wireshark)
- Windows Security & Exploit Awareness
- Threat Analysis & Defensive Techniques
